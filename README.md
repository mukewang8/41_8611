<h3>课程目标：</h3>
<p>虚拟机(VM: Virtual Machine)是通过软件模拟物理机器执行程序的执行器。最初Java语言被设计为基于虚拟机器在而非物理机器，重而实现WORA(一次编写，到处运行)的目的，尽管这个目标几乎被世人所遗忘。所以，<a title="查看与 JVM 相关的文章" target="_blank">JVM</a>可以在所有的硬件环境上执行Java字节码而无须调整Java的执行模式。</p>
